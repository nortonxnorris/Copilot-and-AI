# Data Security and Compliance for AI
![Data Security for AI](Artifacts/DataSecurityAIBanner.png)

## Overview
Microsoft Purview provides a unified data security foundation for AI adoption by helping organizations discover, classify, and protect sensitive data before it’s ever used by AI systems. It extends information protection, DLP, insider risk, and access governance controls across Microsoft Copilot and connected AI applications to prevent data leakage, oversharing, and misuse. With built‑in visibility, policy enforcement, and auditability, Purview enables organizations to adopt AI confidently while maintaining compliance, data boundaries, and trust.
